## Why
当前仅在未登录(missing/invalid)时弹出橙色提示条,已登录时无任何可见状态。用户希望在
页面上**常驻显示登录状态**(已登录 / 未登录),一眼可知当前是否有有效 HF token。

## What Changes
### 1. 顶部常驻登录状态徽章
在页面顶部(header 右侧、设置按钮旁)常驻显示一个登录状态徽章:
- `logged_in` → 绿色「● 已登录(HF token 有效)」
- `invalid`   → 橙色「● token 失效,请重新登录」
- `missing`   → 橙色「● 未登录」
徽章由进入页面的 `/api/token` 检测结果驱动;登录成功后实时更新为已登录。

### 2. 与既有提示条共存
徽章为常驻状态展示;原有的橙色非阻断提示条(missing/invalid 时出现)继续保留,
二者不冲突:徽章表明"当前状态",提示条提供"如何登录"的引导操作。
